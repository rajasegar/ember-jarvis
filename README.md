ember-jarvis
==============================================================================

Ember addon which provides codemod and customization services for AST-Builder, AST-Finder and JARVIS 


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.24 or above
* Ember CLI v3.24 or above
* Node.js v12 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-jarvis
```
